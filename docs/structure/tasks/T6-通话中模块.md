## 开发步骤

### Step 1: 阅读基准画板
读取 `31-通话中展开.json` 和 preview 截图，理解全屏通话中布局：
- 号码/姓名 + 通话时间
- 缩小按钮 `bt_btn_lessen_n.png`
- 键盘按钮 `keyboard_btn_number_keyboard_n.png`
- 静音按钮 `keyboard_btn_number_mute_n.png`
- 切换按钮 `keyboard_btn_number_phone_n.png`
- 挂断按钮 `keyboard_btn_number_hang_up_n.png`

### Step 2: 搭建全屏通话中布局
- 顶部：号码/姓名 + 通话计时 + 缩小按钮
- 底部：键盘/静音/切换/挂断四个功能按钮

### Step 3: 搭建通话中小窗
读取 `32-通话中小框.json`：
- 浮窗背景 `phone_pop_up_1.png`
- 号码 + 通话时间
- 静音按钮 `bt_btn_mute_n.png` + 挂断按钮 `bt_btn_hang_up_n.png`

### Step 4: 实现 DTMF 键盘
读取 `33-通话键盘.json`，在通话中弹出数字键盘：
- 3×4 数字键盘（0-9、*、#），使用键盘样式图层
- 键盘弹出时 `keyboard_btn_number_keyboard_f.png`（按下态图标）
- 底部功能按钮保持不变

### Step 5: 实现顶栏通话状态
读取 `34-通话-顶栏状态.json`：
- 通话小窗隐藏后，顶栏显示通话状态提示条
- `top_phone_display.png` — 顶栏背景
- `top_icon_phone.png` — 电话图标
- 通话时间文字

### Step 6: 实现全屏/小窗/顶栏切换
- 全屏 → 缩小 → 小窗
- 小窗 → 点击 → 全屏
- 小窗 → 隐藏 → 顶栏提示条

### Step 7: 验证
对比五个 preview 截图，确认各形态的布局还原度。
