## 开发步骤

### Step 1: 阅读基准画板
读取 `10-联系人-详情.json` 和 preview 截图，理解弹窗布局：
- 弹窗背景 `bt_linkman_pop_up_1.png`
- 大头像 `icon_linkman_bg_big_3_1.png`
- 号码项背景 `bt_list_pop_up_n_1.png`
- 拨打电话图标 `icon_list_phone_1.png`

### Step 2: 搭建弹窗布局
创建 DialogFragment 或 PopupWindow 布局：
- 弹窗居中显示
- 顶部：联系人大头像 + 姓名
- 下方：电话号码列表（可滚动）
- 每条号码右侧有拨打电话按钮

### Step 3: 设置样式
根据画板图层提取文字样式：
- 联系人姓名样式
- 电话号码样式
- 号码类型标签

### Step 4: 导入切图
- `bt_linkman_pop_up_1.png` / `bt_linkman_pop_up_1_1.png` — 弹窗背景
- `bt_linkman_pop_up_3_1.png` — 两行号码时的弹窗背景
- `icon_linkman_bg_big_3_1.png` / `icon_linkman_big_bg_1.png` — 大头像背景
- `bt_list_pop_up_n_1.png` — 号码项正常背景
- `icon_list_phone_1.png` — 拨打电话图标

### Step 5: 处理交互状态
对比 variant 画板与 base：
- **按下态**（04）：号码项按下时的视觉反馈
- **两行**（05）：弹窗高度适配两条号码，背景切换
- **多条极限**（11）：号码列表可滚动，文字截断/换行处理

### Step 6: 验证
对比四个 preview 截图，确认弹窗在不同号码数量下的布局还原度。
