## 开发步骤

### Step 1: 阅读基准画板
读取 `21-来电展开.json` 和 preview 截图，理解全屏来电布局：
- 背景 `android_bg.png` + `android_bg备份.png`
- 缩小按钮 `bt_btn_lessen_n.png`
- 接听按钮 `keyboard_btn_number_answer_n.png`
- 挂断按钮 `keyboard_btn_number_hang_up_n.png`
- 来电号码/姓名文字

### Step 2: 搭建全屏来电布局
创建 IncomingCallActivity 的全屏布局：
- 顶部：来电号码/姓名 + 缩小按钮
- 底部：接听（绿色）和挂断（红色）两个大按钮
- 全屏背景图

### Step 3: 搭建来电小窗布局
读取 `22-来电小框.json`，创建小窗模式：
- 浮窗背景 `phone_pop_up_1.png`
- 来电号码/姓名文字
- 接听按钮 `bt_btn_answer_calls_n.png`
- 挂断按钮 `bt_btn_hang_up_n.png`

### Step 4: 实现隐私模式
对比 `23-来电小框-隐私模式.json` 和正常小窗：
- 隐私模式隐藏来电号码，仅显示"来电"提示
- 接听/挂断按钮不变

### Step 5: 实现全屏/小窗切换
- 全屏模式下点击缩小按钮 → 切换到小窗
- 小窗模式下点击窗体 → 展开全屏

### Step 6: 导入切图
- `bt_btn_lessen_n.png` — 缩小按钮
- `keyboard_btn_number_answer_n.png` — 接听按钮
- `keyboard_btn_number_hang_up_n.png` — 挂断按钮
- `phone_pop_up_1.png` — 小窗背景
- `bt_btn_answer_calls_n.png` — 小窗接听按钮
- `bt_btn_hang_up_n.png` — 小窗挂断按钮

### Step 7: 验证
对比三个 preview 截图，确认全屏、小窗和隐私模式的布局还原度。
