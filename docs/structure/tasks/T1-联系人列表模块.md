## 开发步骤

### Step 1: 阅读基准画板
读取 `06-联系人.json` base 画板和 preview 截图，理解整体布局：
- 左侧无内容（外壳区域）
- 右侧联系人列表，按姓氏首字母分组
- 最右侧字母索引栏（search_bg）

### Step 2: 搭建联系人列表布局
根据画板中图层 rect 坐标创建 Fragment 布局：
- RecyclerView 实现联系人列表
- 列表项使用 `bt_list_bg_n.png` 作为背景
- 头像使用 `icon_linkman_bg_*.png` 系列
- 列表按姓氏分组，显示分组 header

### Step 3: 设置文字样式
根据图层的 color/fontSize/fontFace 字段设置：
- 联系人姓名：FZLanTingHeiPro，颜色和字号从画板提取
- 分组标题字母

### Step 4: 导入联系人相关切图
- `bt_list_bg_n.png` — 列表项正常背景
- `bt_list_bg_p.png` — 列表项按下背景
- `icon_linkman_bg_*.png` — 联系人头像背景系列
- `search_bg.png` — 右侧字母索引栏背景

### Step 5: 实现字母索引交互
对比 `07-联系人-检索-#～A固定位置.json` 和其他检索 variant：
- 右侧字母索引栏支持触摸滑动
- 触摸时弹出字母气泡（`searching_bar_bg.png` + `search_pop_up.png`）
- #~A 段：气泡固定在索引栏顶部
- Y~Z 段：气泡固定在索引栏底部
- B~X 段：气泡跟随手指位置移动
- 触摸字母时列表滚动到对应分组

### Step 6: 处理按下状态
对比 `01-联系人-按下.json` 和 base：
- 列表项按下时背景切换为 `bt_list_bg_p.png`
- 实现 StateListDrawable 或 selector

### Step 7: 验证
对比 preview 截图，确认列表布局、字母索引交互还原度。
